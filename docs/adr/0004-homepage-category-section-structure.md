# 홈페이지를 카테고리 섹션 구조로 전환

홈페이지의 도구 목록을 단일 평면 그리드에서 카테고리별 섹션으로 전환한다.

PDF 도구와 이미지 도구가 공존하게 되면서 단일 그리드로는 도구 간 성격 차이를 전달하기 어렵다. 섹션 구조를 도입하면 새로운 카테고리(이미지, 텍스트 등)가 추가될 때 홈페이지 리팩토링 없이 섹션만 추가하면 된다.

## Considered Options

- **단일 평면 그리드** — 구조 변경 최소화. 도구가 늘어날수록 카테고리 구분이 불가해져 재리팩토링 필요.
- **카테고리 섹션 구조** ✅ — 섹션 제목 아래 각 카테고리 그리드 배치. 확장에 유리.

## Consequences

`page.tsx`의 `tools` 배열을 카테고리별로 분리한다. 기존 PDF 도구는 "PDF 도구" 섹션으로 묶이고, 증명사진은 "이미지 도구" 섹션의 첫 번째 도구로 추가된다.
