# Tools

브라우저에서 100% 로컬로 실행되는 문서·이미지 처리 도구 모음. 서버 업로드 없이 사용자의 파일을 처리해 프라이버시를 보장한다.

## Language

### 공통

**Tool (도구)**:
사용자가 단일 목적의 파일 처리 작업을 수행하는 페이지 단위 기능.
_Avoid_: 기능, 서비스, 메뉴

**로컬 처리 (Local Processing)**:
파일이 서버에 전송되지 않고 사용자 브라우저 안에서만 처리되는 방식. 이 프로젝트의 핵심 원칙.
_Avoid_: 클라이언트 처리, 프론트엔드 처리

**다운로드 (Download)**:
처리 완료된 결과물을 사용자의 기기에 저장하는 행위. 서버 저장 없음.
_Avoid_: 저장, 내보내기

### PDF 도구

**병합 (Merge)**:
두 개 이상의 PDF 파일을 하나의 PDF로 합치는 작업.
_Avoid_: 합치기, 결합

**분할 (Split)**:
하나의 PDF를 페이지 단위로 나누는 작업.
_Avoid_: 나누기, 잘라내기

**변환 (Convert)**:
PDF를 다른 포맷(이미지 등)으로 바꾸거나, 다른 포맷을 PDF로 바꾸는 작업.
_Avoid_: 내보내기, 저장

**워터마크 (Watermark)**:
PDF 페이지 위에 텍스트 또는 이미지를 반투명하게 덧씌우는 작업.
_Avoid_: 도장, 마크

### 증명사진 도구

**증명사진 (ID Photo)**:
여권·운전면허·입사지원 등 공식 서류에 사용하는 규격화된 인물 사진.
_Avoid_: 프로필 사진, 여권 사진 (용도가 특정될 때만 사용)

**규격 (Spec)**:
특정 용도에서 요구하는 증명사진의 가로×세로 크기(cm 또는 px). 예: 여권 3.5×4.5cm.
_Avoid_: 사이즈, 크기

**배경 제거 (Background Removal)**:
사진에서 인물만 남기고 배경을 투명하게 만드는 처리.
_Avoid_: 누끼, 배경 삭제

**배경색 (Background Color)**:
배경 제거 후 인물 뒤에 채울 단색. 규격별로 허용 색상이 다름 (여권: 흰색만 허용).
_Avoid_: 배경, 바탕색

**얼굴 중앙 배치 (Face Centering)**:
얼굴 감지를 통해 인물의 얼굴이 출력 규격의 중앙 상단에 위치하도록 자동으로 크롭 영역을 설정하는 동작.
_Avoid_: 자동 크롭, 얼굴 감지

**인쇄 레이아웃 (Print Layout)**:
A4 용지 한 장에 동일한 증명사진 여러 장을 배열한 PDF. 편의점 출력을 위한 출력물.
_Avoid_: 인쇄용 PDF, 출력 레이아웃

**미세 조정 (Fine Adjustment)**:
자동 얼굴 중앙 배치 이후 사용자가 드래그로 크롭 영역을 수동으로 수정하는 행위.
_Avoid_: 수동 조정, 크롭 편집
